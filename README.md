# 介绍
### Git全局设置用户凭证:
```
git config --global user.name "your name"
git config --global user.email "your email"
```
### 创建git仓库并提交:
```
mkdir MyWeb-Quick
cd MyWeb-Quick
git init
touch README.md
git add README.md
git commit -m "first commit"
git remote add origin https://gitee.com/xmail978/MyWeb-Quick.git
git push -u origin master
```
